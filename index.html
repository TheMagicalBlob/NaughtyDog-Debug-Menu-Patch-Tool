Naughty Dog Game Debug Mode Enabler

This Tool Can Be Used To Enable The Debug Mode/Menus On Any
PS4 Naughty Dog Game, For Any Title ID (CUSAXXXXX) And On Most Patches.

The Tool Can Either Connect To A PS4 via PS4Debug To Toggle The Debug Mode While The Game's Running,
Or It Can Patch A Decrypted .bin/.elf Executable To Enable The Debug Menu Through A Custom .pkg
